Overview

HPulse does not:

✘ Remember data from months ago forever in HDFS

✘ Do any learning, try to be clever

✘ Modify the cluster

HPulse does:

✓ Present data from minutes ago to now

✓ Provide sub-second sampling to catch transient events

✓ Work with any compatable Hadoop install (see FAQ)

Endpoints

Things HPulse can monitor:

  • Namenode
  • Resource Manager/JobTracker
  • Datanode
  • Node Manager/TaskTracker

Just enter the web address followed by /jmx
